# My-Blogs
This is a web-app that allows the user to create blogs.

## Setup/Installation Requirements
* Fork/Clone the repository
```
   $ git clone https://github.com/andreassenmarvin/My-blogs.git
```
* Open your project on IntelliJ.
* Run the project.
